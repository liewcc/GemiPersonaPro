# 🛡️ Welcome to Asset Sanitizer

Your all-in-one workstation for managing, auditing, and cleaning AI-generated assets.

### 📄 Metadata & Prompt Management
- **Audit & Edit**: Inspect and modify the hidden metadata of your downloaded images.
- **Instant Activation**: Send a saved prompt directly to the Gemini engine to start a new generation instantly.
- **Global Traceability**: Automatically tracks source URLs (Gems) and the local paths of your reference images.

### 🪄 Professional Watermark Removal
The core power of this tool is its multi-tier batch cleaning system:

1.  **Inverse Alpha (Lossless)**: ⚡ **Try this first!** It is the fastest method and perfectly preserves image quality.
2.  **AI Inpainting (LaMa) & Hybrid**: 🧠 **Advanced Cleaning**. Use these if watermarks are offset, if there are multiple logos, or if the standard method leaves artifacts as these methods are more resource-intensive.
3.  **Range Focus**: Don't want to process everything? Target specific sequences (e.g., images `01` to `50`) to save time.
4.  **Manual Refinement**: Personal touch needed? Use the manual tool after a batch run to precisely fix any remaining spots.

### 🚀 How to Start
Use the **Sidebar** on the left to select a **Folder** or a **File**. 
Once selected, this area will transform into your interactive **Image Gallery**.
